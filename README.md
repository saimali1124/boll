# Moviedle 🎬

A Wordle-inspired game where you guess movies in 7 tries!

## Features

- 🎯 Guess the Bollywood movie in 7 attempts
- 🔍 Autocomplete movie suggestions as you type
- 📅 Year range narrowing hints (e.g., "2001 < ? < 2005")
- 🎭 Genre matching hints (shows count and reveals matching genres)
- 🎨 Beautiful, modern UI with gradient backgrounds

## How to Play

1. Start typing a movie name to see suggestions
2. Select a movie from the dropdown
3. Get hints about:
   - Whether the target movie was released earlier or later
   - How many genres match (and which ones)
4. Use the year range display to narrow down your search
5. Win by guessing the correct movie in 7 tries!

## Installation

```bash
npm install
```

## Running the App

```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## Build for Production
